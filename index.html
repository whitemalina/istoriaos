История развития ОС

Операционная система (ОС) - комплекс программ, которые обеспечивают управление аппаратурой ЭВМ, планирование эффективного использования её ресурсов и решение задач по заданиям пользователей.

Перейдите по одной из вкладок, чтобы прочитать историю о ОС:

Windows

Windows — операционная система для ПК и рабочих станций, разработанная корпорацией Microsoft

Mac OS

Mac OS — семейство проприетарных операционных систем производства корпорации Apple

Linux

Linux — семейство Unix-подобных операционных систем на базе ядра Linux
